Função CONCATENAR

Une dois ou mais textos em uma única célula. Ótima para combinar nome e sobrenome, código + descrição, etc.

=CONCATENAR(texto1; texto2; ...)
📋 NOME COMPLETOTexto + Texto
=CONCATENAR(A1;" ";B1)—
Dica moderna: No Excel 2019+ você pode usar o operador & em vez de CONCATENAR: =A1&" "&B1

FN 13

Função ESQUERDA e DIREITA

Extrai um número específico de caracteres do início ou do fim de um texto. Ideal para extrair prefixos, sufixos, códigos, DDD etc.

=ESQUERDA(texto; núm_caract)  ·  =DIREITA(texto; núm_caract)
📋 EXTRAIR CARACTERESEsquerda e Direita
=ESQUERDA(A1;N)—
=DIREITA(A1;N)—
Exemplo: ESQUERDA("(11) 99999-0000"; 4) → "(11)" para extrair o DDD.
FN 14

Função NÚM.CARACT

Conta o número total de caracteres de um texto, incluindo espaços. Útil para validar tamanho de campos como CPF, CEP, senhas.

=NÚM.CARACT(texto)
📋 CONTAR CARACTERESConta inclusive espaços
=NÚM.CARACT(A1) 0 caracteres
0
Validação: Um CPF com máscara tem 14 caracteres: "000.000.000-00".

📋 DECOMPOR UMA DATASelecione uma data
FunçãoResultado

📋 CALCULAR IDADE / TEMPODiferença entre datas
UnidadeResultado

🔢 Funções Matemáticas
FN 17

Função ARRED

Arredonda um número para uma quantidade específica de casas decimais. Essencial para valores monetários e relatórios financeiros.

=ARRED(número; núm_dígitos)
📋 ARREDONDAR VALORCasas decimais ajustáveis
=ARRED(A1;N)—
=ARREDONDAR.PARA.CIMA/BAIXO—
Variações: ARREDONDAR.PARA.CIMA sempre sobe, ARREDONDAR.PARA.BAIXO sempre desce, independente do dígito.
FN 18

Função MOD

Retorna o resto da divisão de um número por outro. Útil para identificar números pares/ímpares ou ciclos periódicos.

=MOD(número; divisor)
📋 RESTO DA DIVISÃOPar ou ímpar?
=MOD(A1;B1) Informe os valores
Truque: =MOD(A1;2)=0 verifica se um número é par. Se o resultado for 0, é par; se for 1, é ímpar.
🔍 Funções de Pesquisa
FN 19

Função PROCV

Procura um valor na primeira coluna de uma tabela e retorna um valor de outra coluna na mesma linha. Uma das funções mais poderosas do Excel.

=PROCV(valor_proc; tabela; núm_coluna; FALSO)
📋 TABELA DE PRODUTOSBusca por código
CódigoProdutoPreço (R$)
P001NotebookR$ 3.500,00
P002MouseR$ 89,90
P003TecladoR$ 199,00
P004MonitorR$ 1.299,00
P005HeadsetR$ 349,00
FórmulaResultado
=PROCV(D1;A:C;2;FALSO) — Produto—
=PROCV(D1;A:C;3;FALSO) — Preço—
Parâmetros: valor a buscar; tabela; número da coluna a retornar; FALSO = correspondência exata.
FN 20

Função SEERRO

Captura erros em fórmulas e retorna um valor alternativo em vez de exibir #N/D, #DIV/0!, #REF! etc. Muito usada em conjunto com PROCV.

=SEERRO(valor; valor_se_erro)
📋 TRATAR ERROS NO PROCVCódigo inválido = mensagem amigável
=SEERRO(PROCV(A1;tabela;2;FALSO);"Produto não encontrado") Informe um código
Prática recomendada: sempre envolva seus PROCVs com SEERRO para evitar que erros apareçam para o usuário final.
📊 Gráficos no Excel
FN 21

Gráfico de Barras / Colunas

Ideal para comparar valores entre categorias. Use para visualizar vendas por mês, desempenho por vendedor ou qualquer comparação direta entre itens.

Inserir → Gráficos → Coluna ou Barra
📊 VENDAS MENSAIS — BARRASEdite os valores e veja o gráfico atualizar
Jan Fev Mar Abr Mai Jun
Quando usar: comparação entre categorias discretas. Prefira colunas para dados temporais e barras horizontais quando os rótulos são longos.
FN 22

Gráfico de Linhas

Perfeito para mostrar tendências ao longo do tempo. Evidencia crescimento, queda e sazonalidade de uma série de dados contínua.

Inserir → Gráficos → Linha
📊 TEMPERATURA MÉDIA — LINHASTendência ao longo do tempo
Jan Fev Mar Abr Mai Jun
Dica visual: use linha suavizada (spline) para dados contínuos e linha reta para dados discretos. No Excel: clique com botão direito → Formatar série → Linha suavizada.
